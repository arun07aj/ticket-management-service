# Getting Started with Create React App

This project has been recently migrated from CRA to Rsbuild.

## Available Scripts

In the project directory, you can run:

### `npm start`

Create a `.env.dev` file under `/webapp` by taking `.env.dev.template` file as a reference.
Runs the app in the development mode.\
Open [http://localhost:3000/tmsapp](http://localhost:3000/tmsapp) to view it in your browser.

### `npm run build:dev`

Builds the app for development to the `build` folder.

### `npm run build:prod`

Builds the app for production to the `build` folder.

### `npm run preview`

Preview the production build locally.

### `npm run lint`

Run ESLint on the source code.
